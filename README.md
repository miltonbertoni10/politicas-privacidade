## Como ativar GitHub Pages

1. Acesse https://github.com/miltonbertoni10/politicas-privacidade/settings/pages
2. Em "Source", selecione "Deploy from a branch"
3. Branch: `main` · Folder: `/ (root)`
4. Salve. Em ~1 minuto a URL fica ativa.

## Como atualizar

Edite o HTML correspondente e dê push. GitHub Pages atualiza automaticamente.
